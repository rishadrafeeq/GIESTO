Update Products Without Coding

Use Google Sheets like Excel. Changes appear on the website after you save the sheet.

Step 1 — Open your product sheet

Ask your developer for the GIESTO Products Google Sheet link. Bookmark it — this is where you add and edit products.

No Google Sheet yet? Download the template, upload it to Google Drive, and open with Google Sheets:

Download CSV template

Step 2 — Edit products in the sheet

Each row is one product. Important columns:

Step 3 — Publish the sheet (one-time setup)

  1. In Google Sheets: File → Share → Publish to web
  2. Choose Comma-separated values (.csv)
  3. Click Publish and copy the link
  4. Send the link to your developer — they connect it once

After setup, you only edit the sheet. No code, no uploads to the website.

Add a new product

  1. Add a new row at the bottom of the sheet
  2. Fill every column (use a similar product as a guide)
  3. Set category to formal, casual, or sports
  4. Set active to yes
  5. Save — refresh the collection page on the website

Hide or remove a product

Set active to no — it disappears from the shop but stays in the sheet for later.
